📖 使用說明書
🩺 核心功能
  • 記錄每日 ☀️ 早上與 🌙 晚上的血壓數值
  • 點擊列表右側的「✏️」即可隨時補填或修改
  • 每次點擊儲存，系統將完全自動進行雲端安全覆蓋
  • 一鍵產出乾淨的 PDF，方便回診給醫師看
☁️ 雲端備份教學
  1. 前往 GitHub 註冊免費帳號並登入
  2. 點選右上角個人頭像 → 選擇 Settings
  3. 左側選單滑到最下方 → 點選 Developer settings
  4. 展開 Personal access tokens → 點選 Tokens (classic)
  5. 點擊右上方 Generate new token → ⚠️ 務必在下拉選單選擇 Generate new token (classic)
  6. Note 欄位填寫「血壓備份」
  7. 期限 (Expiration) 選 No expiration
  8. 權限清單 (Scopes) 找到並只勾選 gist
  9. 滑至最底部點擊 Generate token，產生後立刻複製該串密碼，貼回此 App 即可連線！
