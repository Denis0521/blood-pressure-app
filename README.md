# 血壓記錄 APP (PWA 版本)

這是一個具有 3D 立體高質感的血壓記錄 Progressive Web App (PWA)。

## 部署與安裝方式
1. 註冊並登入 [GitHub](https://github.com/)。
2. 建立一個新的 Repository (儲存庫)。
3. 將本資料夾內的 `index.html`, `manifest.json`, `sw.js`, `icon.svg` 全數上傳。
4. 到該專案的 **Settings** -> **Pages**，將 Source 設為 `main` 分支並儲存。
5. 幾分鐘後，用手機瀏覽器打開 GitHub Pages 提供的網址，即可看見畫面。
6. 瀏覽器下方會彈出「**加到主畫面**」或「**安裝應用程式**」，點擊後即可在桌面獲得專屬 APP 圖標！
